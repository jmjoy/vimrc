# Basic

* 将`jk`映射为 `<Esc>`，这个太重要了，兼容输入法中英文切换 （`<Shift>`）

* 映射`<Space>`到`<Leader>`，`<Leader>y` `<Leader>d` `<Leader>p` 访问系统寄存器

* 将 `<C-h>` `<C-j>` `<C-k>` `<C-l>` 作为窗口移动方向键

* 将 `H` `J` `K` `L` 作为Easy-Motion插件的方向键，因为这个确实太常用了

* 将 `W` `E` 作为标签页移动方向键

* `J` 和 `K` 的功能暂时用 `\` 和 `?` 代替
